# GCSE Mathematics Marking Guidance

## Mark Scheme Abbreviations

| Code | Term | Definition |
|------|------|------------|
| **M** | Method Mark | Awarded for correct mathematical method, not lost for purely numerical errors |
| **P** | Process Mark | Awarded for a correct process as part of a problem solving question |
| **A** | Accuracy Mark | Awarded after a correct method or process; if no method or process is seen then full marks for the question are implied but see individual mark schemes for more details |
| **C** | Communication Mark | Awarded for a fully correct statement(s) with no contradiction or ambiguity |
| **B** | Unconditional Accuracy Mark | No method needed |
| **oe** | Or Equivalent | Accept alternative correct forms of the answer |
| **cao** | Correct Answer Only | The answer must be exactly as specified (no partial credit) |
| **ft** | Follow Through | When appropriate as per mark scheme |
| **sc** | Special Case | Special marking case (see specific criterion guidance) |
| **dep** | Dependent | This mark depends on a previous mark being awarded |
| **indep** | Independent | This mark can be awarded independently of other marks |
| **awrt** | Answer Which Rounds To | Accept answers that round to the specified value |
| **isw** | Ignore Subsequent Working | Once correct answer is seen, ignore any further working |
